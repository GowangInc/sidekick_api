# Icon Files Needed

The extension requires three icon files:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

You can:
1. Create simple colored squares with any image editor
2. Use an online icon generator
3. Design custom icons for your extension

For now, you can use any PNG images of the correct sizes as placeholders.
